Multiple Data Sources

This report demonstrates combining more than one data source into one report.

The root data source for the report is a list of states in the United States. A sub-level is defined that pulls data from a separate data source, a list of companies in the S&P 500 index, and links to the outer data source (states) by matching the headquarters location of the company. The result is a list of states with each state containing a list of the companies located in that state.
