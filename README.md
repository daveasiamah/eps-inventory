# EPS IMS WEB

> FrontEnd Web version of the EPS Inventory Management System.

## Quick Start

```bash
# Install dependencies
npm install

# Serve on localhost:3000
npm start

# Build for production
npm run build
```

## App Info

Inventory Management in Node.js and React.
